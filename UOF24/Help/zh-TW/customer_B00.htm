主檔維護

客戶類別維護
點選左方樹狀的節點 [ 客戶類別 ] ，按一下上方功能列的 [ 新增類別 ] ，即會出現新增類別對話視窗
輸入「類別名稱」，設定完後按下 [ 確定 ]，即可新增一筆客戶類別資料
點選要修改的「類別名稱」，即會出現修改類別對話視窗，可修改類別資料，修改完後按下 [ 確定 ]，即完成修改
勾選要刪除的類別資料，按一下上方功能列的  [ 刪除類別 ]，詢問是否確定刪除，點選 [ 確定]，即可刪除勾選的類別
注意：已被客戶資料使用的客戶類別，將無法被勾選刪除
客戶等級維護
點選左方樹狀的節點 [ 客戶等級 ] ，按一下上方功能列的 [ 新增等級 ] ，即會出現新增等級對話視窗
輸入「等級名稱」、「等級」，設定完後按下 [ 確定 ]，即可新增一筆客戶等級資料
最高等級：99；最低等級：1
點選要修改的「等級名稱」，即會出現修改等級對話視窗，可修改等級資料，修改完後按下 [ 確定 ]，即完成修改
勾選要刪除的等級資料，按一下上方功能列的  [ 刪除等級 ]，詢問是否確定刪除，點選 [ 確定 ]，即可刪除勾選的等級
注意：已被客戶資料使用的客戶等級，將無法被勾選刪除

地區別維護
點選左方樹狀的節點 [ 地區別 ] ，按一下上方功能列的 [ 地區別維護 ] ，即會出現地區別維護對話視窗
按一下上方功能列「新增地區別」，輸入地區別名稱，即可新增一筆地區別資料
如要修改地區別名稱，選取要修改的地區別節點，雙擊二下地區別即可進行修改
如要刪除地區別，先選取要刪除的地區別節點，按一下上方功能列 [ 刪除地區別 ] ，即可刪除地區別及其下的子地區
完成地區別的維護後，按一下 [ 關閉視窗 ]，即完成地區別維護
注意：已被客戶資料使用的地區別，將無法被刪除
