查詢客戶資料

可從不同角度來查詢客戶資料，並快速獲得查詢結果。

特殊查詢
提供查詢 [ 一週內發佈的客戶 ]、 [ 未設定資料歸屬的客戶 ]、 [ 未設定聯絡人的客戶 ]
選取 [ 特殊查詢 ] 條件下拉選單的選項，畫面上會顯示符合條件的客戶資料清單
[ 一週內發佈的客戶 ]：顯示一週內已發佈的客戶資料
[ 未設定資料歸屬的客戶 ]：顯示所有未設定資料歸屬的未發佈客戶資料
[ 未設定聯絡人的客戶 ]：顯示所有未設定聯絡人的已發佈及未發佈客戶資料
注意：畫面上顯示的已發佈客戶資料，是個人具有瀏覽權限的客戶資料
關鍵字查詢
根據輸入的「關鍵字」，查詢出符合條件的客戶資料
輸入要查詢的關鍵字，點選 [ 查詢 ]
畫面上會顯示客戶代號、客戶簡稱、客戶名稱、統一編號或備註，查詢符合關鍵字的已發佈客戶資料
注意：畫面上顯示的已發佈客戶資料，是個人具有瀏覽權限的客戶資料
依類別查詢
依客戶類別來查詢客戶資料
點選左方畫面上的 [ 類別 ] 標籤，左下方顯示所有客戶類別樹狀節點
根據選取的類別節點，右方畫面會顯示，此類別中的已發佈客戶資料及新增未發佈客戶資料清單
注意：畫面上顯示的已發佈客戶資料，是個人具有瀏覽權限的客戶資料
依行業別查詢
依行業別來查詢客戶資料
點選左方畫面上的 [ 行業別 ] 標籤，左下方顯示所有的行業別及無設定行業別客戶的樹狀節點
根據點選的行業別節點，右方畫面會顯示，此行業別中的已發佈客戶資料及新增未發佈客戶資料清單
點選無設定行業別客戶的節點，右方畫面會顯示，無設定行業別的已發佈客戶資料及新增未發佈客戶資料清單
注意：畫面上顯示的已發佈客戶資料，是個人具有瀏覽權限的客戶資料
依地區別查詢
依地區別來查詢客戶資料
點選左方畫面上的 [ 地區別 ] 標籤，左下方顯示所有的地區別及無設定地區別客戶的樹狀節點
根據點選的地區別節點，右方畫面會顯示，此地區別中的已發佈客戶資料及新增未發佈客戶資料清單
點選地區別父節點，顯示的客戶資料包含其下地區別子節點的客戶資料
點選無設定地區別客戶的節點，右方畫面會顯示，無設定地區別的已發佈客戶資料及新增未發佈客戶資料清單
注意：畫面上顯示的已發佈客戶資料，是個人具有瀏覽權限的客戶資料
