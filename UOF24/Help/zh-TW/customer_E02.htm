如何刪除客戶資料

刪除未發佈及已發佈的客戶資料。

單筆刪除未發佈的客戶資料
點選要刪除的未發佈客戶資料的「客戶代號」，即會出現修改客戶資料對話視窗
客戶資料狀態為未發佈，畫面下方會顯示 [ 發佈 ] 及 [ 刪除 ] 按鈕
點選 [ 刪除 ] 按鈕，提示訊息詢問是否確定要刪除，點選 [ 確定 ]，即刪除未發佈的客戶資料
完成後，畫面上將不會顯示已刪除的客戶資料
注意：未發佈的客戶資料，只有建立者才可以刪除客戶資料

單筆刪除已發佈的客戶資料
點選要刪除的已發佈客戶資料的「客戶代號」，即會出現修改客戶資料對話視窗
客戶資料狀態為已發佈，畫面下方將不會顯示 [ 發佈 ] 按鈕，會根據權限顯示 [ 刪除 ] 按鈕
登入者必須具有刪除或資料管理者權限才會顯示 [ 刪除 ] 按鈕
點選 [ 刪除 ] 按鈕，提示訊息詢問是否確定要刪除，點選 [ 確定 ]，即刪除已發佈的客戶資料
完成後，畫面上將不會顯示已刪除的客戶資料
注意：已發佈的客戶資料，必須具有刪除或資料管理者權限才能刪除
批次刪除客戶資料
勾選要刪除的客戶資料，按一下上方功能列的 [ 刪除客戶 ]
詢問是否確定刪除，點選 [ 確定]，即可進行刪除勾選的客戶資料(含已發佈及未發佈)
如果沒有刪除已發佈客戶資料的權限，會出現 "您沒有刪除客戶資料權限" 的訊息，將無法刪除勾選的已發佈客戶資料；如果有勾選要刪除的未發佈客戶資料，仍會被刪除
完成後，畫面上將不會顯示已被刪除的客戶資料
注意：未發佈的客戶資料，只有建立者才可以刪除客戶資料；已發佈的客戶資料，必須具有刪除或資料管理者任一權限才能刪除
