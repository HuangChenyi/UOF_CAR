問卷統計

設定各問卷統計結果的觀看權限，觀看回收結果、統計圖表及匯出問卷回收資料EXCEL。

設定觀看權限
進入 [ 問卷統計 ] 畫面，只有具有問卷管理者或問卷製作者的權限，才能設定問卷觀看權限
問卷製作者，只能設定自己所製作的問卷觀看權限
點選 [ 設定觀看權限 ] ， 出現設定觀看權限的對話視窗
選取人員完成後，按下 [ 確定 ] 按鈕，即完成該問卷觀看權限的設定
具有該問卷統計結果瀏覽權限的人員，當公佈時間一到，即使問卷統計結果不公開，也能看到問卷的統計圖表及回收結果
具有瀏覽全部統計資料權限的人員，能觀看全部問卷的統計圖表及回收結果
問卷填寫者，只有在問卷統計結果公開的情形下，才能在個人問卷清單中觀看統計圖表
觀看回收結果
進入 [ 問卷統計 ] 畫面，選擇要觀看回收結果的問卷，按一下問卷資料後方的 [ 回收結果 ]，即會出現問卷回收結果對話視窗
顯示全部答案選項：有勾選，在瀏覽已填寫的問卷資料時，將會顯示題目所有的選項；沒有勾選，只會顯示填寫者選擇的選項
再次通知未填寫者：可對未填寫問卷的人員，再次發送問卷填寫通知
如果是記名問卷，回收結果的「人員姓名」會顯示問卷填寫者姓名
要查看記名問卷的內容，按一下 [ 查看內容 ]，即會出現瀏覽填寫問卷內容的對話視窗，顯示填寫者所填寫的內容
如果是不記名問卷，只會顯示回收問卷字樣，將不會顯示問卷填寫者姓名
要查看不記名問卷的內容，按一下「問卷」，即會出現瀏覽填寫問卷內容的對話視窗，顯示該份問卷的填寫內容
觀看統計圖表
點選 [ 統計圖表 ]，會出現統計圖表對話視窗，分為長條圖統計、圓餅圖統計及投票者分析
長條圖統計畫面，使用長條圖來顯示問卷中各題目選項的得票數及百分比
點選長條圖統計畫面上各題目的 [ 交叉分析 ]，出現交叉分析的對話視窗，顯示的圖表是依問卷填寫人屬性來分析填寫者選擇的選項比例
只有題目的選項類型為單選及複選才能進行交叉分析
圓餅圖統計畫面，使用圓餅圖來顯示問卷中各題目選項的得票數及百分比
投票者分析畫面，分析目前已完成問卷填寫的填寫人屬性分析
匯出EXCEL
選擇[直接匯出EXCEL]，提示訊息詢問是否確定產生問卷回收資料檔案，點選 [ 確定 ]，系統會於5分鐘內將文件產生至 {下載匯出資料} 畫面中
選擇[排程匯出EXCEL]，會出現提示訊息詢問是否確定產生問卷回收資料檔案，點選[確定]，系統會於5分鐘內將文件產生至{下載匯出資料}畫面中
點選{下載匯出資料}畫面，所有曾經匯出的文件會列表於此，也可以再次下載使用；如果不需要時，則可以勾選文件進行刪除
點選「檔名」，即可以下載EXCEL文件
只有問卷製作者及問卷管理者，才能點選[直接匯出EXCEL]和[排程匯出EXCEL]按鈕，匯出問卷回收資料檔案
