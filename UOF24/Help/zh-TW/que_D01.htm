如何建立問卷

新增問卷，設定問卷基本資料、問卷題目及設定調查時間/對象。

設定問卷基本資料
進入 [ 問卷維護 ]  畫面，按一下上方功能列的 [ 新增問卷 ]， 即會出現新增問卷對話視窗
選取問卷來源，分為：新增問卷及從其他問卷複製，如果點選從其他問卷複製，可以挑選現有的問卷來複製問卷內容
注意：可選擇複製的問卷，必須為允許他人複製的問卷
點選新增問卷，顯示問卷基本設定畫面
問卷狀態：製作中，表示正在製作的問卷，尚未開始進行問卷調查
問卷統計結果：

「公開」表示公開問卷統計的結果，問卷製作者、填寫者及具有觀看問卷統計結果者，皆可觀看問卷結果

「不公開」表示不公開問卷統計的結果，只有問卷製作者及具有觀看問卷統計結果者，才能觀看問卷結果
問卷記名：

「記名」表示回收的問卷會記名，觀看回收結果時，會顯示該問卷的填寫者

「不記名」表示回收的問卷不會記名，觀看回收結果時，無法知道是誰填寫問卷

是否允許他人複製此問卷：

「是」表示當此問卷啟用後，允許其他問卷製作者進行複製

「否」表示當此問卷啟用後，不允許其他問卷製作者進行複製
填寫問卷時是否提供儲存：

「是」表示問卷填寫者在填寫問卷時，能儲存問卷內容，暫不送出問卷

「否」表示問卷填寫者在填寫問卷時，無法暫存問卷內容，只能直接送出問卷
是否允許重複填寫：

「是」一張問卷可以寫很多次，沒有限定次數送出問卷

「否」一張問卷只能寫一次
是否允許填寫者修改或取消：

「是」在問卷調查時間截止前，填寫者都可修改或取消個人填寫過的問卷

「否」問卷只要填寫了就不允許修改或取消
設定完成後，點選 [ 儲存 ]，即完成問卷基本資料設定
注意：必須完成問卷基本設定，才能進行題目設定
設定問卷題目 - 新增題目
問卷題目來源，分為：新增題目及匯入題目
按一下上方功能列的 [ 新增題目 ] ，即會出現新增題目對話視窗
選項類型：分為單選、複選、單行文字、多行文字、摘要說明及日期欄位
是否必填：「是」表示為必填題目；「否」表示為非必填題目
選項排列方式：

只有選項類型為單選或複選時，才能選擇選項的排列方式。

「水平」表示選項以水平的方式排列(可設定一列幾個選項)；「垂直」表示選項以垂直的方式排列(一列一個選項)
行數：只有選項類型為多行文字時，才能輸入行數
輸入「題目」，選擇 「選項類型」，點選 [ 確定 ]，完成題目新增，如果選項類型是單選或複選，畫面下方會出現維護選項功能
增加其他選項，供使用者輸入意見：勾選後，選項中將會增加一個 [其他] 的選項及文字方塊
按下方功能列的 [ 新增選項 ] ，即會出現新增選項對話視窗，輸入「內容」，點選 [ 確定 ]，即完成新增
分數：此選項設定的分數(必須大於等於0)
選項中增加空白欄位，供使用者輸入：勾選後，選項後方將會增加一個空白的文字方塊，供使用者輸入
點選要修改的「選項」，即會出現修改選項的對話視窗，可修改選項資料，修改完後按下 [ 確定 ]，即完成修改
勾選要刪除的選項資料，按一下功能列的 [ 刪除選項 ]，詢問是否確定刪除，點選 [ 確定]，即可刪除勾選的選項資料
點選 [ 確定 ]，題目設定畫面上顯示新增的題目及選項
設定問卷題目 - 匯入題目
按一下上方功能列的 [ 匯入題目 ] ，即會出現匯入題目對話視窗
選擇問卷類別及問卷，勾選要複製的問卷題目，點選 [ 確定 ]，完成題目匯入
注意：可選擇複製題目的問卷，必須為允許他人複製的問卷，則問卷題目才會顯示且能複製
設定問卷題目 - 刪除選擇題目
勾選要刪除的問卷題目，按一下上方功能列的 [ 刪除選擇題目 ]
會詢問是否確定刪除，點選 [ 確定 ]，即會刪除勾選的題目
設定問卷題目 - 修改題目順序
按一下上方功能列的 [ 修改題目順序 ] ，即會出現修改題目順序對話視窗
選取題目，點選，選取的題目順序將會置頂；選取題目，點選，選取的題目順序將會置底；
選取題目，點選，選取的題目順序將會上移；選取題目，點選，選取的題目將會下移
點選 [ 確定 ]，完成題目順序調整
設定問卷題目 - 設定填寫人屬性
按一下上方功能列的 [ 設定填寫人屬性 ]，即會出現設定填寫人屬性對話視窗
點選 [ 屬性 ] 欄位下的屬性名稱，會出現屬性值維護的對話視窗，可維護屬性值資料
點選 [ 確定 ]，完成問卷填寫人屬性值維護
可逐一勾選問卷要設定的屬性值，或者點選 [ 全選 ]，該屬性的全部屬性值即會全部勾選，點選 [ 確定 ]，完成問卷填寫人屬性設定
設定問卷調查時間/對象
完成問卷題目設定後，按一下問卷維護畫面上方的 {設定調查時間/對象}，會顯示設定問卷調查時間及對象的畫面
選擇「問卷調查時間起迄」：問卷調查的起迄時間，問卷調查時間開始，才能填寫問卷；超過問卷調查結束時間，將無法填寫問卷
選擇「統計結果下架日期」：停止在填寫者的個人首頁公佈問卷統計結果的日期
注意：當問卷統計結果設定為公開時，會在填寫者的個人首頁上公佈問卷統計結果，此選項才有作用
選擇「統計結果公佈時間」：開始公佈問卷統計結果的時間
注意：當問卷統計結果設定為公開時，填寫者才能觀看問卷的統計結果，此選項才有作用
選擇「問卷調查對象」：可填寫問卷的人員
是否發填寫問卷調查通知：是否發送問卷調查通知給問卷調查對象
當問卷啟用後，如果有修改問卷調查時間，會發送問卷調查修改通知
當問卷啟用後，可以新增或刪除問卷調查對象。
會對新增的調查對象發送問卷調查通知；被刪除的調查對象，會發送問卷調查取消通知，且會修改問卷目前的發送數及回收數
點選 [ 儲存 ]，新增完成製作中的問卷

關於啟用問卷的方法，請參閱如何啟用、中止及關閉問卷調查
