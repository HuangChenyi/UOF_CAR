如何修改客戶資料

修改未發佈及已發佈的客戶資料。

修改未發佈的客戶資料
點選要修改的「客戶代號」，即會出現修改客戶資料對話視窗
輸入要修改的客戶資料後，點選 [ 儲存 ]，完成修改客戶資料
資料狀態：「未發佈」表示客戶資料未被發佈，只有建立者才可以修改或刪除未發佈的客戶資料
資料歸屬：此筆客戶資料發佈後，具有瀏覽此筆客戶資料權限的瀏覽者
欄位其他1~6：可填寫紀錄客戶相關資料的欄位
客戶資料狀態為未發佈，畫面下方會顯示 [ 發佈 ] 及 [ 刪除 ] 按鈕
發佈：發佈客戶資料，如果此筆客戶資料尚未設定聯絡人資料，則會顯示訊息詢問是否確定要發佈，點選 [ 確定 ]，即完成客戶資料的發佈
刪除：可將此筆未發佈的客戶資料予以刪除
點選畫面上方 [ 聯絡人資料 ] 標籤，進入聯絡人資料維護畫面
因為客戶資料尚未發佈，所以客戶資料建立者可繼續維護聯絡人資料(新增、修改、刪除聯絡人資料)
維護完成後，點選 [ 關閉視窗 ]，畫面上顯示修改後的客戶資料
注意：未發佈的客戶資料，只有建立者才可以修改、刪除客戶資料及維護聯絡人資料
修改已發佈的客戶資料
點選要修改的「客戶代號」，即會出現修改客戶資料對話視窗
輸入要修改的客戶資料後，點選 [ 儲存 ]，完成修改客戶資料
資料狀態：「已發佈」表示客戶資料已經發佈，必須具有修改權限，才能修改已發佈的客戶資料；必須具有刪除或資料管理者任一權限，才能修改或刪除已發佈的客戶資料
資料歸屬：具有瀏覽此筆客戶資料權限的瀏覽者
欄位其他1~6：可填寫紀錄客戶相關資料的欄位
客戶資料狀態為已發佈，畫面下方將不會顯示 [ 發佈 ] 按鈕，會根據權限顯示 [ 刪除 ] 按鈕
刪除：可將此筆已發佈的客戶資料予以刪除，必須具有刪除或資料管理者任一權限，才能對已發佈的客戶資料進行刪除
點選畫面上方 [ 聯絡人資料 ] 標籤，進入聯絡人資料維護畫面
因為客戶資料狀態為已發佈，因此要維護聯絡人資料必須具有對應權限
新增聯絡人：具有新增、修改、刪除或資料管理者任一權限即可新增聯絡人
修改聯絡人：具有修改、刪除或資料管理者任一權限即可修改聯絡人
刪除聯絡人：具有刪除或資料管理者任一權限即可刪除聯絡人
維護完成後，點選 [ 關閉視窗 ]，畫面上顯示修改後的客戶資料
注意：要修改已發佈的客戶資料，除了要具有修改、刪除或資料管理者權限外，也必須是該筆資料的瀏覽者，才能看到此筆客戶資料
系統管理者角色的人可以由聯絡人建立外部會員帳號功能。
