屬性維護

維護問卷填寫人屬性資料。

進入 [ 填寫人屬性維護 ] 畫面，按一下上方功能列的 [ 新增屬性 ] ，即會出現新增屬性對話視窗
輸入「屬性名稱」，選擇「屬性類型」，按一下功能列的 [ 增加屬性值 ] ，下方即會出現屬性值樹狀節點
屬性類型：「單選」表示只能選擇一個選項；「多選」表示可勾選多個選項
輸入「屬性值名稱」，即可新增一筆屬性值資料
如要刪除屬性值資料，勾選要刪除的屬性值，按一下功能列的 [ 刪除屬性值 ]，會詢問是否確定刪除，點選 [ 確定 ]，即可刪除勾選的屬性值
設定完屬性資料後按下 [ 確定 ]，即可新增一筆屬性資料
點選要修改的「屬性名稱」，即會出現修改屬性對話視窗，可修改屬性資料，修改完後按下 [ 確定 ]，即完成修改
如要刪除屬性資料，勾選要刪除的屬性，按一下功能列的 [ 刪除屬性 ]，會詢問是否確定刪除，點選 [ 確定 ]，即可刪除勾選的屬性
