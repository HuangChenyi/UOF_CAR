如何新增客戶資料

新增客戶資料及聯絡人資料。

新增客戶資料
進入 [ 客戶資料維護 ] 畫面，請點選左方畫面上 [ 類別 ]、[ 行業別 ] 或 [ 等級 ] 任一標籤，具有新增、修改、刪除資料權限或資料管理者權限的使用者，即可點選上方功能列按紐
按一下上方功能列的 [ 新增客戶 ] ，即會出現新增客戶資料對話視窗
輸入「客戶代號」、「客戶簡稱」、「客戶名稱」等客戶資料，選取客戶所屬的「客戶類別」，並設定「資料歸屬」權限
資料狀態：
「未發佈」表示客戶資料未被發佈，只有建立者才可以修改或刪除未發佈的客戶資料
「已發佈」表示客戶資料已經發佈，必須具有修改權限，才能修改已發佈的客戶資料；必須具有刪除或資料管理者任一權限，才能修改或刪除已發佈的客戶資料
資料歸屬：此筆客戶資料被發佈後，具有瀏覽此筆客戶資料權限的瀏覽者
欄位其他1~6：可填寫紀錄客戶相關資料的欄位
注意：必須儲存客戶資料後，才能維護聯絡人資料
設定完成後，點選 [ 儲存 ]，即完成新增客戶資料
新增客戶資料完成儲存後，畫面下方會顯示 [ 發佈 ] 及 [ 刪除 ] 按鈕
發佈：發佈客戶資料，如果此筆客戶資料尚未設定聯絡人資料，則會顯示訊息詢問是否確定要發佈，點選 [ 確定 ]，即完成客戶資料的發佈
刪除：可將此筆未發佈的客戶資料予以刪除
新增聯絡人資料
點選新增客戶資料畫面上方的 [ 聯絡人資料 ] 標籤，進入維護聯絡人資料畫面
按一下上方功能列的 [ 新增聯絡人 ] ，即會出現新增聯絡人對話視窗
輸入「聯絡人姓名」等資料，點選 [ 確定 ]，完成新增，聯絡人資料清單顯示新增的聯絡人資料
點選要修改的「聯絡人姓名」，即會出現修改聯絡人對話視窗，可修改聯絡人資料，修改完後按下 [ 確定 ]，即完成修改
勾選要刪除的聯絡人資料，按一下上方功能列的 [ 刪除聯絡人 ]，詢問是否確定刪除，點選 [ 確定]，即可刪除勾選的聯絡人資料
點選 [ 關閉視窗 ]，畫面上顯示新增的客戶資料
