如何删除客户数据

删除未发布及已发布的客户数据。

单笔删除未发布的客户数据
点选要删除的未发布客户数据的“客户代号”，即会出现修改客户数据对话窗口
客户数据状态为未发布，画面下方会显示 [ 发布 ] 及 [ 删除 ] 按钮
点选 [ 删除 ] 按钮，提示消息询问是否确实要删除，点选 [ 确定 ]，即删除未发布的客户数据
完成后，画面上将不会显示已删除的客户数据
注意：未发布的客户数据，仅创建者才可以删除客户数据

单笔删除已发布的客户数据
点选要删除的已发布客户数据的“客户代号”，即会出现修改客户数据对话窗口
客户数据状态为已发布，画面下方将不会显示 [ 发布 ] 按钮，会根据权限显示 [ 删除 ] 按钮
登录者必须具有删除或数据管理者权限才会显示 [ 删除 ] 按钮
点选 [ 删除 ] 按钮，提示消息询问是否确实要删除，点选 [ 确定 ]，即删除已发布的客户数据
完成后，画面上将不会显示已删除的客户数据
注意：已发布的客户数据，必须具有删除或数据管理者权限才能删除
批次删除客户数据
勾选要删除的客户数据，单击上方功能列的 [ 删除客户 ]
询问是否确定删除，点选 [ 确定]，即可进行删除勾选的客户数据(含已发布及未发布)
如果没有删除已发布客户数据的权限，会出现 "您没有删除客户数据权限" 的消息，将无法删除勾选的已发布客户数据；如果有勾选要删除的未发布客户数据，仍会被删除
完成后，画面上将不会显示已被删除的客户数据
注意：未发布的客户数据，仅创建者才可以删除客户数据；已发布的客户数据，必须具有删除或数据管理者任一权限才能删除
