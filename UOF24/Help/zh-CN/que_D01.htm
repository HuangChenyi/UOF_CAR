如何创建问卷

新建问卷，设置问卷基本数据、问卷题目及设置调查时间/对象。

设置问卷基本数据
进入 [ 问卷维护 ]  画面，单击上方功能列的 [ 新建问卷 ]， 即会出现新建问卷对话窗口
选定问卷源，分为：新建问卷及从其他问卷复制，如果点选从其他问卷复制，可以挑选现有的问卷来复制问卷内容
注意：可选择复制的问卷，必须为允许他人复制的问卷
点选新建问卷，显示问卷基本配置画面
问卷状态：制作中，表示正在制作的问卷，尚未开始进行问卷调查
问卷统计结果：

“公开”表示公开问卷统计的结果，问卷制作者、填写者及具有观看问卷统计结果者，皆可观看问卷结果

“不公开”表示不公开问卷统计的结果，仅问卷制作者及具有观看问卷统计结果者，才能观看问卷结果
问卷记名：

“记名”表示回收的问卷会记名，观看回收结果时，会显示该问卷的填写者

“不记名”表示回收的问卷不会记名，观看回收结果时，无法知道是谁填写问卷

是否允许他人复制此问卷：

“是”表示当此问卷启用后，允许其他问卷制作者进行复制

“否”表示当此问卷启用后，不允许其他问卷制作者进行复制
填写问卷时是否提供保存：

“是”表示问卷填写者在填写问卷时，能保存问卷内容，暂不送出问卷

“否”表示问卷填写者在填写问卷时，无法暂存问卷内容，只能直接送出问卷
是否允许重复填写：

“是”一张问卷可以写很多次，没有限定次数送出问卷

“否”一张问卷只能写一次
是否允许填写者修改或取消：

“是”在问卷调查时间截止前，填写者都可修改或取消个人填写过的问卷

“否”问卷只要填写了就不允许修改或取消
配置完成后，点选 [ 保存 ]，即完成问卷基本数据设置
注意：必须完成问卷基本配置，才能进行题目设置
设置问卷题目 - 新建题目
问卷题目源，分为：新建题目及导入题目
单击上方功能列的 [ 新建题目 ] ，即会出现新建题目对话窗口
选项类型：分为单选、复选、单行文字、多行文字、摘要说明及日期字段
是否必填：“是”表示为必填题目；“否”表示为非必填题目
选项排列方式：

仅选项类型为单选或复选时，才能选择选项的排列方式。

“水平”表示选项以水平的方式排列(可设置一列几个选项)；“垂直”表示选项以垂直的方式排列(一列一个选项)
行数：仅选项类型为多行文字时，才能输入行数
输入“题目”，选择 “选项类型”，点选 [ 确定 ]，完成题目新建，如果选项类型是单选或复选，画面下方会出现维护选项功能
增加其他选项，供用户输入意见：勾选后，选项中将会增加一个 [其他] 的选项及文字方块
按下方功能列的 [ 新建选项 ] ，即会出现新建选项对话窗口，输入“内容”，点选 [ 确定 ]，即完成新建
分数：此选项设置的分数(必须大于等于0)
选项中增加空白字段，供用户输入：勾选后，选项后方将会增加一个空白的文字方块，供用户输入
点选要修改的“选项”，即会出现修改选项的对话窗口，可修改选项数据，修改完后按下 [ 确定 ]，即完成修改
勾选要删除的选项数据，单击功能列的 [ 删除选项 ]，询问是否确定删除，点选 [ 确定]，即可删除勾选的选项数据
点选 [ 确定 ]，题目设置画面上显示新建的题目及选项
设置问卷题目 - 导入题目
单击上方功能列的 [ 导入题目 ] ，即会出现导入题目对话窗口
选择问卷类别及问卷，勾选要复制的问卷题目，点选 [ 确定 ]，完成题目导入
注意：可选择复制题目的问卷，必须为允许他人复制的问卷，则问卷题目才会显示且能复制
设置问卷题目 - 删除选择题目
勾选要删除的问卷题目，单击上方功能列的 [ 删除选择题目 ]
会询问是否确定删除，点选 [ 确定 ]，即会删除勾选的题目
设置问卷题目 - 修改题目顺序
单击上方功能列的 [ 修改题目顺序 ] ，即会出现修改题目顺序对话窗口
选定题目，点选，选定的题目顺序将会置顶；选定题目，点选，选定的题目顺序将会置底；
选定题目，点选，选定的题目顺序将会上移；选定题目，点选，选定的题目将会下移
点选 [ 确定 ]，完成题目顺序调整
设置问卷题目 - 设置填写人属性
单击上方功能列的 [ 设置填写人属性 ]，即会出现设置填写人属性对话窗口
点选 [ 属性 ] 字段下的属性名称，会出现属性值维护的对话窗口，可维护属性值数据
点选 [ 确定 ]，完成问卷填写人属性值维护
可逐一勾选问卷要设置的属性值，或者点选 [ 全部选定 ]，该属性的全下属性值即会全部勾选，点选 [ 确定 ]，完成问卷填写人属性设置
设置问卷调查时间/对象
完成问卷题目设置后，单击问卷维护画面上方的 {设置调查时间/对象}，会显示设置问卷调查时间及对象的画面
选择“问卷调查时间起迄”：问卷调查的起迄时间，问卷调查时间开始，才能填写问卷；超过问卷调查退出时间，将无法填写问卷
选择“统计结果下架日期”：停止在填写者的个人首页公布问卷统计结果的日期
注意：当问卷统计结果设置为公开时，会在填写者的个人首页上公布问卷统计结果，此选项才有作用
选择“统计结果公布时间”：开始公布问卷统计结果的时间
注意：当问卷统计结果设置为公开时，填写者才能观看问卷的统计结果，此选项才有作用
选择“问卷调查对象”：可填写问卷的人员
是否发填写问卷调查通知：是否发送问卷调查通知给问卷调查对象
当问卷启用后，如果有修改问卷调查时间，会发送问卷调查修改通知
当问卷启用后，可以新建或删除问卷调查对象。
会对新建的调查对象发送问卷调查通知；被删除的调查对象，会发送问卷调查取消通知，且会修改问卷当前的发送数及回收数
点选 [ 保存 ]，新建完成制作中的问卷

关于启用问卷的方法，请参阅如何启用、中止及关闭问卷调查
