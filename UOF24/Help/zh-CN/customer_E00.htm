如何新建客户数据

新建客户数据及联络人数据。

新建客户数据
进入 [ 客户数据维护 ] 画面，请点选左方画面上 [ 类别 ]、[ 行业别 ] 或 [ 等级 ] 任一标签，具有新建、修改、删除数据权限或数据管理者权限的用户，即可点选上方功能列按纽
单击上方功能列的 [ 新建客户 ] ，即会出现新建客户数据对话窗口
输入“客户代号”、“客户简称”、“客户名称”等客户数据，选定客户所属的“客户类别”，并设置“数据归属”权限
数据状态：
“未发布”表示客户数据未被发布，仅创建者才可以修改或删除未发布的客户数据
“已发布”表示客户数据已经发布，必须具有修改权限，才能修改已发布的客户数据；必须具有删除或数据管理者任一权限，才能修改或删除已发布的客户数据
数据归属：此笔客户数据被发布后，具有浏览此笔客户数据权限的浏览者
字段其他1~6：可填写纪录客户相关数据的字段
注意：必须保存客户数据后，才能维护联络人数据
配置完成后，点选 [ 保存 ]，即完成新建客户数据
新建客户数据完成保存后，画面下方会显示 [ 发布 ] 及 [ 删除 ] 按钮
发布：发布客户数据，如果此笔客户数据尚未设置联络人数据，则会显示消息询问是否确实要发布，点选 [ 确定 ]，即完成客户数据的发布
删除：可将此笔未发布的客户数据予以删除
新建联络人数据
点选新建客户数据画面上方的 [ 联络人数据 ] 标签，进入维护联络人数据画面
单击上方功能列的 [ 新建联络人 ] ，即会出现新建联络人对话窗口
输入“联络人姓名”等数据，点选 [ 确定 ]，完成新建，联络人数据列表显示新建的联络人数据
点选要修改的“联络人姓名”，即会出现修改联络人对话窗口，可修改联络人数据，修改完后按下 [ 确定 ]，即完成修改
勾选要删除的联络人数据，单击上方功能列的 [ 删除联络人 ]，询问是否确定删除，点选 [ 确定]，即可删除勾选的联络人数据
点选 [ 关闭窗口 ]，画面上显示新建的客户数据
