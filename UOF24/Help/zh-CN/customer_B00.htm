主档维护

客户类别维护
点选左方树状的节点 [ 客户类别 ] ，单击上方功能列的 [ 新建类别 ] ，即会出现新建类别对话窗口
输入“类别名称”，设置完后按下 [ 确定 ]，即可新建一笔客户类别数据
点选要修改的“类别名称”，即会出现修改类别对话窗口，可修改类别数据，修改完后按下 [ 确定 ]，即完成修改
勾选要删除的类别数据，单击上方功能列的  [ 删除类别 ]，询问是否确定删除，点选 [ 确定]，即可删除勾选的类别
注意：已被客户数据使用的客户类别，将无法被勾选删除
客户等级维护
点选左方树状的节点 [ 客户等级 ] ，单击上方功能列的 [ 新建等级 ] ，即会出现新建等级对话窗口
输入“等级名”、“等级”，设置完后按下 [ 确定 ]，即可新建一笔客户等级数据
最高档级：99；最低档级：1
点选要修改的“等级名”，即会出现修改等级对话窗口，可修改等级数据，修改完后按下 [ 确定 ]，即完成修改
勾选要删除的等级数据，单击上方功能列的  [ 删除等级 ]，询问是否确定删除，点选 [ 确定 ]，即可删除勾选的等级
注意：已被客户数据使用的客户等级，将无法被勾选删除

地区别维护
点选左方树状的节点 [ 地区别 ] ，单击上方功能列的 [ 地区别维护 ] ，即会出现地区别维护对话窗口
单击上方功能列“新建地区别”，输入地区别名称，即可新建一笔地区别数据
如要修改地区别名称，选定要修改的地区别节点，双击二下地区别即可进行修改
如要删除地区别，先选定要删除的地区别节点，单击上方功能列 [ 删除地区别 ] ，即可删除地区别及其下的子地区
完成地区别的维护后，单击 [ 关闭窗口 ]，即完成地区别维护
注意：已被客户数据使用的地区别，将无法被删除
