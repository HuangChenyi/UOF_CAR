如何修改客户数据

修改未发布及已发布的客户数据。

修改未发布的客户数据
点选要修改的“客户代号”，即会出现修改客户数据对话窗口
输入要修改的客户数据后，点选 [ 保存 ]，完成修改客户数据
数据状态：“未发布”表示客户数据未被发布，仅创建者才可以修改或删除未发布的客户数据
数据归属：此笔客户数据发布后，具有浏览此笔客户数据权限的浏览者
字段其他1~6：可填写纪录客户相关数据的字段
客户数据状态为未发布，画面下方会显示 [ 发布 ] 及 [ 删除 ] 按钮
发布：发布客户数据，如果此笔客户数据尚未设置联络人数据，则会显示消息询问是否确实要发布，点选 [ 确定 ]，即完成客户数据的发布
删除：可将此笔未发布的客户数据予以删除
点选画面上方 [ 联络人数据 ] 标签，进入联络人数据维护画面
因为客户数据尚未发布，所以客户数据创建者可继续维护联络人数据(新建、修改、删除联络人数据)
维护完成后，点选 [ 关闭窗口 ]，画面上显示修改后的客户数据
注意：未发布的客户数据，仅创建者才可以修改、删除客户数据及维护联络人数据
修改已发布的客户数据
点选要修改的“客户代号”，即会出现修改客户数据对话窗口
输入要修改的客户数据后，点选 [ 保存 ]，完成修改客户数据
数据状态：“已发布”表示客户数据已经发布，必须具有修改权限，才能修改已发布的客户数据；必须具有删除或数据管理者任一权限，才能修改或删除已发布的客户数据
数据归属：具有浏览此笔客户数据权限的浏览者
字段其他1~6：可填写纪录客户相关数据的字段
客户数据状态为已发布，画面下方将不会显示 [ 发布 ] 按钮，会根据权限显示 [ 删除 ] 按钮
删除：可将此笔已发布的客户数据予以删除，必须具有删除或数据管理者任一权限，才能对已发布的客户数据进行删除
点选画面上方 [ 联络人数据 ] 标签，进入联络人数据维护画面
因为客户数据状态为已发布，因此要维护联络人数据必须具有映射权限
新建联络人：具有新建、修改、删除或数据管理者任一权限即可新建联络人
修改联络人：具有修改、删除或数据管理者任一权限即可修改联络人
删除联络人：具有删除或数据管理者任一权限即可删除联络人
维护完成后，点选 [ 关闭窗口 ]，画面上显示修改后的客户数据
注意：要修改已发布的客户数据，除了要具有修改、删除或数据管理者权限外，也必须是该笔数据的浏览者，才能看到此笔客户数据
系统管理者角色的人可以由联络人创建外部会员帐号功能。
