查询客户数据

可从不同角度来查询客户数据，并快速获得查询结果。

特殊查询
提供查询 [ 一周内发布的客户 ]、 [ 未配置数据归属的客户 ]、 [ 未设置联络人的客户 ]
选定 [ 特殊查询 ] 条件下拉菜单的选项，画面上会显示符合条件的客户数据列表
[ 一周内发布的客户 ]：显示一周内已发布的客户数据
[ 未配置数据归属的客户 ]：显示所有未配置数据归属的未发布客户数据
[ 未设置联络人的客户 ]：显示所有未设置联络人的已发布及未发布客户数据
注意：画面上显示的已发布客户数据，是个人具有浏览权限的客户数据
关键字查询
根据输入的“关键字”，查询出符合条件的客户数据
输入要查询的关键字，点选 [ 查询 ]
画面上会显示客户代号、客户简称、客户名称、统一编号或备注，查询符合关键字的已发布客户数据
注意：画面上显示的已发布客户数据，是个人具有浏览权限的客户数据
依类别查询
依客户类别来查询客户数据
点选左方画面上的 [ 类别 ] 标签，左下方显示所有客户类别树状节点
根据选定的类别节点，右方画面会显示，此类别中的已发布客户数据及新建未发布客户数据列表
注意：画面上显示的已发布客户数据，是个人具有浏览权限的客户数据
依行业别查询
依行业别来查询客户数据
点选左方画面上的 [ 行业别 ] 标签，左下方显示所有的行业别及无设置行业别客户的树状节点
根据点选的行业别节点，右方画面会显示，此行业别中的已发布客户数据及新建未发布客户数据列表
点选无设置行业别客户的节点，右方画面会显示，无设置行业别的已发布客户数据及新建未发布客户数据列表
注意：画面上显示的已发布客户数据，是个人具有浏览权限的客户数据
依地区别查询
依地区别来查询客户数据
点选左方画面上的 [ 地区别 ] 标签，左下方显示所有的地区别及无设置地区别客户的树状节点
根据点选的地区别节点，右方画面会显示，此地区别中的已发布客户数据及新建未发布客户数据列表
点选地区别父节点，显示的客户数据包含其下地区别子节点的客户数据
点选无设置地区别客户的节点，右方画面会显示，无设置地区别的已发布客户数据及新建未发布客户数据列表
注意：画面上显示的已发布客户数据，是个人具有浏览权限的客户数据
