属性维护

维护问卷填写人属性数据。

进入 [ 填写人属性维护 ] 画面，单击上方功能列的 [ 新建属性 ] ，即会出现新建属性对话窗口
输入“属性名称”，选择“属性类型”，单击功能列的 [ 增加属性值 ] ，下方即会出现属性值树状节点
属性类型：“单选”表示只能选择一个选项；“多选”表示可勾选多个选项
输入“属性值名称”，即可新建一笔属性值数据
如要删除属性值数据，勾选要删除的属性值，单击功能列的 [ 删除属性值 ]，会询问是否确定删除，点选 [ 确定 ]，即可删除勾选的属性值
设置完属性数据后按下 [ 确定 ]，即可新建一笔属性数据
点选要修改的“属性名称”，即会出现修改属性对话窗口，可修改属性数据，修改完后按下 [ 确定 ]，即完成修改
如要删除属性数据，勾选要删除的属性，单击功能列的 [ 删除属性 ]，会询问是否确定删除，点选 [ 确定 ]，即可删除勾选的属性
