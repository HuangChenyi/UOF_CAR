Private message

Private message contains messages sent out by the system or other users, for example, meeting notification and notification for things assigned, etc.

Inbox
Click ¡§Message¡¨ to enter ¡§Inbox¡¨; when there is new message, the ¡§Message¡¨ icon on the upper right corner will flash.
Inbox will list messages sent out by system or other users, the user can read, reply or delete it, or search it through key words.
Click the subject of the message to read, then you can enter to see the sender, time and content. If you want to reply the message, you can select ¡§Message replay¡¨, then enter the reply content at the content part, then click ¡§Submit¡¨, the message will then be replied.
You can also forward the message. Click ¡§Forward message¡¨ to forward the message to other users.
If the received message is important work, please click ¡§Convert to task¡¨, then enter the subject and content, then enter the starting and ending time, you can then record this message into the calendar.
Send box
The message in the sent folder is the record left in the system after the user has sent the message.
You can check the message reading time and replying time from the receiver.
Note: When the private message is forwarded to others, you can enclose an attachment document in the mail.
Send new message
Click ¡§send new message¡¨ to enter screen for adding new message, then select member to receive message to decide how many persons to send the message. Meanwhile, the subject of the message must be entered, and the content can be edited through the editing function.
Click ¡§Submit¡¨, the message will be sent out immediately, and the sent message will be recorded in ¡§sent folder¡¨.
Classifying and archiving
Click ¡§Maintain Category¡¨ to set up your own classification category, then click ¡§New category¡¨, then enter category name. After finishing the setup, then click ¡§Exit¡¨ to finish.
In archiving process, please check first the message item in the list, then go to ¡§File as¡¨ to select the category to be archived, the archiving action is then finished. In the future, you can search related information according to your own setup classification.

Note:The messages in ¡§Message¡¨ are mostly the messages sent out by the system to notify the users, hence, lots of message will be accumulated there, the user must remember to clean the inbox folder constantly.
