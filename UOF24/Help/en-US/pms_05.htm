Basic setting of project management

System administrator can authorize the right to some members. Member with different authority will have different operation content.

Member who can create project: This member can only create project, change the draft project he created, and check the project he has created.

Default notification category: When it is checked, the system will send out the notification automatically. When creating new project, the system default values will be those checked items, for those not checked, the system will not send out notifications to them.
