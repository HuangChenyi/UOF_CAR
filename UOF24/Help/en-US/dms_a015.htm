Watermark setup

Make watermark setup on this directory.

Use text watermark

Set up first watermark content. You can select “Downloader’s Name and Department” or “Customize text watermark”, and watermark text will be displayed in PDF file.

Then set up angle to turn, text color and font size of watermark, finally, set up the display location of watermark.

You can preview the effect of watermark. If the setup is finished, please click “Confirm” to finish.

Use of pictorial watermark

Set up first the picture of watermark. Select the picture file to be used as watermark.

Then set up “angle to turn” of picture, finally, set up the display location of the picture.

You can preview the effect of watermark. After finishing the setup, please click “Confirm” to finish.

Display system date: The viewing and printing date will be displayed on the lower left corner of the document.

Display document path: The document path will be displayed at the lower left corner of the document.

Note: Text watermark and pictorial watermark can exist at the same time.


使用文字封面
勾選“使用文件封面”、點選“系統預設”，會呈現「文件管理→浮水印設定」所設定的內容；
勾選“使用文件封面”、點選“自訂封面”，可自行編輯內容，而所需的欄位屬性代號透過複製貼上來使用[系統屬性]與[自訂屬性]；
若文件公佈審核是使用電子簽核模式，可勾選“顯示簽核資訊”，文件封面便會呈現表單簽核明細；
完成上述設定後，可選擇預覽方式(直向或橫向)觀看文件封面，最後再按一下 [確定] 即可。

註：自訂屬性的欄位請參考文件屬性設定。
