Personal form

In general function interface, the user can, through this function, enter all the form information that is related to an individual. It includes personal form, form application, form retrieval, setup of substitute person, and definition of self-made flow. The user selects the tab to be used according to the need, and the user can also click the function to be used from the BPM menu.


Note: For the operation method, please refer to all the operation manuals.
