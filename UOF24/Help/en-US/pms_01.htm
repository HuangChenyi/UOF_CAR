Project modification

Project modification is to seize all the projects macroscopically and to make in-time correction on the project according to the setup.

New project

We provide 3 ways to create a new project: Excel file import, Brand new, Copy from an exist project.
Click ¡§Next¡¨, Input the required data, then click ¡§Submit¡¨ .
You can create phases or tasks for a project:
Phase: click ¡§New Phase¡¨, input the required data in "Phase setup" window, then click ¡§Submit¡¨.
Task: click ¡§New Task¡¨, input the required data in "Task setup" window, then click ¡§Submit¡¨.
You can also add phases or tasks into a phase, click the icon in the ¡§opreate¡¨ field to add phases and tasks.
Use ¡§Move phase/task¡¨ to move up and down the sequence, drag and draw the phase/task to change the sequence.
Start up the project: click the ¡§Project start up¡¨ to start up the project. If the field "Approved by" is not empty in "Project setup" window, the ¡§Project start up¡¨ will show as ¡§Project send for approval¡¨, click the button, system will popup confirmation massage, click ¡§Yes¡¨ for approval, and project will start up as soon as it is approved.

Project modification

After project is started up, the task owner needs to edit on the task progress for each phase in the project report of personal calendar or home page. When a task is finished, when you select ¡§Completed¡¨ and schedule 100%, then in ¡§Project modification¡¨ page, green checked marked will appear in front of the completed phase. System will follow the previous setup to report to the phase manager the schedule of that phase either automatically or manually.

Phase manager can, on his home page, review the completed task in PMS-Audit. If it is not passed, comments need to be entered, then return it to the task owner until the audit is passed.

After all the tasks in one phase are completed, project manager needs to audit on that phase. If it is not passed, it needs to be returned to the phase manager for improvement. In the mean time, in front of this phase, the system will add automatically the word¡§improvement¡¨. 

After finishing all the phases and after passing the audit, project manager will close the project. Then the project will be moved to ¡§Closed¡¨. The project can be found out through query by project, and you can click project name to view.

After starting up project, you can still add new phase and task, and once the phase is completed, it cannot be changed.

Project stop

The so-called project stop means that when a project is carried out to certain extent and cannot be continued or does not need to be continued due to certain reason, project manager/project auditor/admin can together pass the termination action to end this project. 

Once a project stop, it will be unable to be continued, however, if it is needed, the project can be added again for its realization.
