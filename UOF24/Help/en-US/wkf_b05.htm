Parallel flow design

When the form, at certain site, needs more than two sub-flows to carry out approval operation together, it is then needed to set up parallel flow.

In parallel flow, it is not allowed to do adding approval or returning approval. When all the sub-flows in the parallel flow are approved, the form will then be sent to the next site.

Maintain Category
Category must be set up first before parallel flow can be set up. Click ¡§maintain category¡¨ to enter editing screen, then click ¡§New category¡¨ to enter category name.
New parallel flow
It is needed to add more than one sub-flow before parallel flow can be added.
Click the category of parallel flow, then click ¡§New parallel flow¡¨, then enter the flow name, then click ¡§save information¡¨ to finish the setup.
In the screen setting up parallel flow, click ¡§New¡¨, then select the sub-flows to be approved at the same time.
Change or delete parallel flows
You can make change or deletion on the setup parallel flow. Click first on the name of parallel flow to be changed, then click ¡§Parallel flow setup¡¨ to change; however, when you click ¡§Delete parallel flow¡¨, you can then delete the parallel flow. However, the parallel flow in use cannot be deleted.
Note: 1. When parallel flow is added, sub-flow needs to be set up first, please refer to sub-flow design.
