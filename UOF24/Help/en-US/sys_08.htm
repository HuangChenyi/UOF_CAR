External member

It is to manage the group and member at the external side of the company and to maintain corporate organizational structure. Meanwhile, tree structure is used to set up the subordinate relationship among groups. In the member part, you can set up the basic data of the member, and the management of the group and member is planned and maintained by the system administrator.

Maintain group
Press once ¡§Group¡¨ to enter the edit action, and you can double-click it to change the group name; if there is node within the group, then the group should not be deleted.
If sub-group are to be set up, please select first the father group, then press ¡§New group¡¨ to set up sub-group underneath the father group. The hierarchy of the group is not limited; hence, if the group is to be moved, it is OK to use dragging way.
The setup group will be listed in ¡§External organizational chart¡¨ and will be expanded in the form of tree structure.
Maintain category
Member can be set up with different categories to facilitate management and search.
Press once ¡§Category¡¨ to enter and edit, then press ¡§New Category¡¨ to set up the category. You can also use dragging way to move the group.
New member
First, in ¡§External organization chart¡¨, select the group to be added with member.
Then click ¡§New¡¨ to enter member management screen, then set up first the account number and name, for other basic information, please set up according to actual need.
If the user needs to use USB Key for validation, please check ¡§Use USB Key validation¡¨. Then put USB Key into USB slot, then press ¡§read¡¨, if the USB Key validation is not installed, the system will ask you to install, after the installation, it can then read USB Key number, the USB Key can then be handed to the user.
After finishing the setup, press ¡§Submit¡¨, if next data is to be added, you can press ¡§Continue¡¨.
  Note: You can set up whether the member information is to be displayed to the general user according to the field. You can also self-make the name of the field. For detail method, please refer to:Profile-field setup
Member move
When the member is to be switched to different group, you can select ¡§Move¡¨ to enter member movement screen, then click themember and drag it to different group to finish.
Press once ¡§Del.¡¨ to delete the member.
Delete selected member
Click first the group, then check the member. You can select several or all at one time.
Press once ¡§Del.¡¨ to delete the member.
Import Excel member

When the system is to be entered with large batch ofmember data, you can use import way to enter the data to save time.

Click department ¡§Import¡¨. Download first example table, click ¡§Download excel template¡¨, then enter the content according to the example format, after finishing it, please save it.
Click again department ¡§Import¡¨, then press ¡§select¡¨ to select the file, then press ¡§submit¡¨ to import the member.
Import AD member

When the system is to be entered with large batch of AD member, you can use import method to save the data to save time.

Click department ¡§Import(AD)¡¨, it will then be connected to AD Server for checking the member. You can select several or allmember in one time.
Press once ¡§Del.¡¨ to delete themember.
Search member and group

Enter first key words to be searched, after the data entering, if you want to search department, please press ¡§Search Group¡¨, if you want to search members, please press ¡§Search members¡¨.
