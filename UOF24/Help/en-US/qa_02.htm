Add and modify discussion topic article

Each discussion board will be open to certain employees to different authority, and the employees can post related topics on different discussion boards, and they can also modify or delete their own articles.

Add new discussion topic article

Under forum, select discussion board, then click ¡§new topic¡¨ to enter the editing screen of the discussion topic.

In the field, enter discussion topic and discussion content, in the content part, you can use the function in the menu bar to edit the screen, for example, font size, color, editing and table, etc.

If file needs to be uploaded, you can click ¡§select¡¨, then select the file to upload it. You can upload several attachments at one time; uploaded attachments will be listed at the bottom of the article.

After finishing editing the topic, click ¡§Submit¡¨ to add one new discussion topic.

Modify discussion topic or reply an article

The user can only modify his added topic or reply. Click ¡§modify¡¨ within the article to enter and edit the screen.

You can modify the topic, content and attachment, then click ¡§Submit¡¨ to finish revising the discussion topic.


Note:

When add and modify article, the display name will be set up together with the ID of the poster of the discussion board, however, the user name for uploading the attachment will not be displayed in the nickname board and anonymous board.

¡§Display name¡¨: Name entered when person is added and it cannot be changed.

¡§Display nickname¡¨: It is forum nickname in personal setup and it can be changed. If it is not set up, then original name will be displayed.

¡§Anonymous¡¨: All the data will be hidden. Any place for name display will only show "Anonymous".

Delete discussion topic or replied article

The user can only delete his added topic or reply. Within the article, please click ¡§delete¡¨, then click ¡§Submit¡¨ to finish.


Note:

How to add the uploaded photo into the content of the article?

Upload the photo first, the photo will then be listed at the bottom of the photo, then drag the photo to the content editing area, then adjust the photo size to finish.
