Import files in batch

Document can be uploaded and import into the document management system in batch basis so that document management can be put online quickly in the early setup stage.

 

Enter the account and password of UOF system, then drag the file to the document or directory in the directory of ¡§document management system¡¨in the right side.

It can be uploaded according to document or directory, it can be set up with author according to the document, or the version of the document can be set up. After confirmation, please press import to finish.
