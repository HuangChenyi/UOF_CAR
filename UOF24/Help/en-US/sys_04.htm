Organization calendar

Company and department can set up its own calendar. Since each department has different working hours, if it is necessary, you can define for yourself.

Working hours

In ¡§Working hours¡¨, select the department to be set up first. Then click any department in ¡§Department list¡¨ at the left side.

If you want to set up again the working hours of the department, please cancel the checking of ¡§Use previous Department setup¡¨, then set up labor hours and working weeks.

In ¡§authorization level setup¡¨, select first the department to be set up. Then click any department in ¡§department list¡¨ at the left side.
