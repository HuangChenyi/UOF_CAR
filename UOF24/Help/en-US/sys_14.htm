Profile-field setup

You can follow the field to set up if the content of themember information is to be displayed to the general user for viewing. You can also make the field name by yourself.

Maintain internal staff field
Click ¡§Maintain Internal Staff field¡¨ to enter and set up the displayed screen and to check the name of the field to be displayed. Only checked field can be checked by other users; meanwhile, you can also set up field that the user is allowed to change, hence, the user will be allowed to change the checked field.
System will provide six self-made displayed fields, and the administrator can set up the displayed name for each field, for example, graduation school, etc. The user can follow these fields to enter his personal data.
Maintain external member
Click ¡§Maintain external member¡¨ to enter and set up the displayed screen. Check first the fields to be displayed, and only those fields checked can be viewed by other internal users. You can also set up the fields that the users are allowed to change, hence, the users can change on those fields checked.
The system will provide six self-made displayed fields. Administrator can set up displayed name for each field, for example, product purchasing, etc., and the member can follow these fields to enter his own data.
