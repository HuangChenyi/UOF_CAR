Statistics

It is to set up the view authority of the questionnaire statistical result, to set up the viewing of results on returns and statistical graph and to set up the exporting of questionnaire returned EXCEL.

Set up view authority

Enter ¡§Statistics¡¨ screen, only the member with the authority of questionnaire administrator or questionnaire issuer can set up the questionnaire view authority.

Questionnaire issuer can only set up the view authority for his own prepared questionnaire.

Click ¡§View authority setup¡¨, dialogue box for setting up view authority will then appear.

After finishing the task, please click ¡§Submit¡¨ button to finish the setup of the questionnaire view authority.

The person which is set as Viewable can view the Statics and Result of the Survey as soon as the Survey is issued, even the Statistical result is set as Secret.

Member with authority to view all the statistical data can view all the questionnaire statistical graph and results on returns.

Questionnaire answerer, only under the condition when questionnaire statistical result is public, can view statistical graph on personal questionnaire list.

Viewing results on returns

Enter ¡§Statistics¡¨ screen to select questionnaire for viewing results on returns, then click ¡§Results on returns¡¨ at the rear side of questionnaire data, dialogue window of questionnaire results on returns will then appear.

Display all selectable answers: When it is checked, then when you browse the answered questionnaire data, it will display all selectable answers; when it is not checked, only the items selected by the answerer will be displayed.

Resend notice to members who have not yet filled in the questionnaire: to member not answering questionnaire, you can send again questionnaire filling notice.

If it is registered questionnaire, the ¡§Member name¡¨ in results on returns will display the name of the questionnaire answerer.

If you want to check the content of registered questionnaire, please click ¡§Examine content¡¨, a dialogue window for browsing the filled questionnaire content will then appear to display the content filled by the answerer.

If it is anonymous questionnaire, it will only display the letter of ¡§Return questionnaire¡¨, and the name of questionnaire answerer will not be displayed.

If you want to check anonymous questionnaire content, please click ¡§Questionnaire¡¨, dialogue window for browsing the filled questionnaire content will then appear to display the filled content of the questionnaire.

View statistical graph

Click ¡§Statistical graph ¡§, dialogue window for statistical graph will then appear. It can be divided into bar graph statistics, pie graph statistics and voter¡¦s analysis.

In bar graph statistics screen, bar graph is used to display the number of ballots and percentage of each question in questionnaire.

Click ¡§Cross analysis¡¨ in each question on bar graph statistics screen, a dialogue window for cross analysis will then appear. The displayed graph follows the answerer attribute of questionnaire to analyze the selection item percentage selected by the answerer.

Only when the question has selection type of single selection and multi selection, can cross analysis be carried out.

In pie graph statistics screen, pie graph is used to display the number of ballots and percentage of each question item in questionnaire.

In the voter¡¦s analysis screen, it will carry out answerer¡¦s attribute analysis on the currently filled questionnaire.
