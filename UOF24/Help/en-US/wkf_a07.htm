Set up customized flow

When the administrator set up certain site of the form flow as ¡§customized flow¡¨, then when the applicant or reviewer is at ¡§customized Stop¡¨ , it is needed to set up the flow before it can be sent out.

Add new and maintain the self-made flow
In the form approval, click ¡§Define Customized flow¡¨ , then click ¡§New¡¨ or select ¡§Select Customized Flow¡¨, the setup screen will then show up.
If you click ¡§New¡¨ , you will then enter the screen for editing the flow, then enter the flow name and click ¡§Save¡¨ .
Start to set up the site, click ¡§insert stop¡¨, and the steps and descriptions are as in the followings:
Can this site be modified: For each stop, whether the approver is allowed to self-change the approving member of the next site or to self-change the member to be notified after send-out can be set up.
Approval Type : Type for this Stop.
  • General : One approver, 'Approve' to next Stop, 'Reject' to end the Flow.
  • Multi-OR : More than one approvers, 'Approve' by anyone to next Stop, 'Reject' to end the Flow will be according to Multi-or approval setting in [Maintain form].
  • Multi-AND : More than one approvers, 'Approve' by all to next Stop, 'Reject' by anyone to end the Flow.
Delay setup: Each stop can be set up with time-out time, hence, if it is not approved after time-out, the system will send out message to follow up the approver.
Approver: Approval can select general approval member.
Notified member:Member that should be notified when forms are sent to this stop.
Notify member after approved: They are member to be notified after form approval.
Click ¡§Submit¡¨ to add a new stop or flow, however, if you want to modify, please click ¡§Setup¡¨ to modify the stop or flow.
Duplicate Customized flow
Click first the name of the customized flow to be duplicated, then click ¡§Copy¡¨ to completely duplicate the entire flow.

Note: In customized flow, you can add several new approval stops, and the user can customized the flow direction.
