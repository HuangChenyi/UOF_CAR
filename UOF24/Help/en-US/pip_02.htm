Forgot password

System will update the user password as the random code generated by the system.
The system will send the password to the user (Email registered in the system).
The user uses this password to log in the system.
At this moment, system will ask the user to change the password one more time.

Note: The gender, name, birthday, date employee took office and educational background in the personal data area is maintained by the system administrator, the user cannot change it arbitrarily.
