Vote

Enterprise can make a voting survey so that employees can vote for some topics.

Change voting content
If the voting is not started yet or if there is no one join the voting, then the voting topic content can be changed, however, if somemember are doing the voting, then it should not be changed.
Click issue contents to perform the issue change.
Raise new vote
Start a new vote is to restart for an ended voting issue.
Click the ended issue content, then press ¡§Raise new vote¡¨ to reset the content, time and vote member, etc.
