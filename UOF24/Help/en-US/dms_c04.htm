Maintain Document

System administrator and Document administrator can Move, Destory, Void all documents in batch. Directory administrator can only do the above opertions in managed directories.

Document move
Move Directory : Drag&Draw the directory to the directory you want to move.
New Sub-directory : Click a directory first, then press once ¡§New sub-directory¡¨ to add a new one.
Delete Directory : Check the directory and Click [Delete Directory] to delete.
Click [Rearrange] to Rearrange.
Document movement maintenance
In the ¡§directory¡¨ at the left side, click the directory, then go to the document list at the right side to check document to be moved, you can select several documents in one time.
Press again ¡§Document move¡¨, then click the directory for the document to be moved into, then press ¡§document move¡¨ , the movement action is then completed.
Document destroyed
Click directory at the ¡§directory¡¨ on the left side, then go to document list at the right side to check document to be destroyed. You can select several documents at one time.
Then press once ¡§Document destroyed¡¨ to finish document destroying action.
Document voided
Click directory on the ¡§directory¡¨ at the left side, then go to the document list at the right side to check document to be cancelled. You can select several documents at one time.
Press again ¡§Document voided¡¨ to finish document cancelling action.
Change author
You can change the author of the document in entire lot basis. Select the documents to be changed from the directory or status, then press ¡§Change author¡¨ to select which author to be changed to finish.
Move or destroy document according to the status
Select ¡§status¡¨ on the left side, then select from ¡§status¡¨ menu, which includes: Released, invalid, void, off shelved and release reject, etc.
After selecting the status, you can select ¡§Document move¡¨ or ¡§Document destroyed, that is, the voided or off shelved document is to be destroyed.
Note:

Destroying: To really destroy all the versions of the document, and the system will not have this document at all.

Voiding: The reader will be unable to use this document anymore, but the author still can be downloaded for checking.
