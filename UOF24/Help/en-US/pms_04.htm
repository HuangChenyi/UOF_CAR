Query

By project

You can make query from the project point of view, and you can find project related to you.

Click ¡§by project¡¨ to select the project status, and project status includes all status, not yet started, in progress, stopped and closed. You can make a query on the project by the status, and you can also enter the project name and click ¡§Query¡¨ to finish.

Click the project name in the project list to check the project information, approval history, project Gantt chart, meanwhile, you can also do EXCL exporting on the project.

Project manager can click the icon within ¡§action¡¨ of project list, to stop project that is not yet started or project that is in progress, or to delete project that is closed or stopped. Moreover, project manager can also click the task name within the project to remind or follow up task that is not yet started or is overdue, and related task manager will then receive the system message. Project participants can check the participated project.

By task

Query from the task point of view, and you can find tasks that are related to yourself.

Click ¡§By task¡¨, then select first the project status. Statuses can be divided into five types, which are respectively all status, not yet started, in progress, stopped and close. You can select the status according to the task to be queried.

Next, select the starting and ending time of ¡§Time Frame¡¨, then select the project name, and the system default is all projects. Then select project participants, which can also be the participating department, and this is a needed selection.

After all conditions are set up, click ¡§Query¡¨, system will then list the project queried.

Click the project name to check the related information. Approval history and project Gantt chart of the project, meanwhile, you can use EXCEL to export the project information. Click the task name under project, you can also check and see ¡§report on history record¡¨ and ¡§approval history¡¨, meanwhile, you can also remind or follow up on task under different status.
