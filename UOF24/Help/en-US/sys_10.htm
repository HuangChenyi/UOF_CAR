Password principle:

Principle for not activating enhanced password framework (System default)

1. For new account, the password is defaulted as blank.

2. Personalized password must have at least 6 characters.

Principle to activate enhanced password framework

1. Default password generated by system automatically.

2. The password is a mix of English and number.

3. Password should be at least eight digits.

When enhanced type password principle is activated, as long as newmember is added, the system will provide automatically password to admin and the new user; if the user is to be input in batch basis, password letter can be sent in batch way. It is only needed to click ¡§Mail Password¡¨.
