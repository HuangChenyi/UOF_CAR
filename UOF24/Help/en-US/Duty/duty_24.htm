Import Travel Subsidy
If there are different Travel Subsidy in organizations or Departments, you can import for each organization or department.
Select year after clicking ¡iImport Travel Subsidy¡j, and upload the excel file, there is a template file for reference.
Generate Travel Subsidy
Check the setup in Leave>Configuration>Travel Subsidy, all Seniority is already setup the relevant amount of Travel Subsidy.
Select year after click ¡iGenerate Travel Subsidy¡jthen click ¡iSubmit¡jfor all employees. Error Message will display if some employees can't generate the Subsidy, but others will still successfully generated.
Travel Subsidy Maintain
You can select an individual employee to modify the Travel Subsidy, but those which had applied the Travel Subsidy can't be modified.

Remark : Before generating Travel Subsidy, check the setup in Leave>Configuration>Travel Subsidy, all Seniority is already setup the relevant amount of Travel Subsidy. Please refer to Travel Subsidy.
