How to maintain other parameters of configuration

The maximum of the overtime work set up by Labor Standards Law as well as the daily work hour setup can be completed within this parameter.

Maintain parameters
The maximum overtime per month for male and female can be set up according to Labor Standards Law. When the total overtime hours in the duty overtime work form have exceeded the maximum, then the supervisor will see reminder information showing that this employee has exceeded overtime hours this month when the supervisor is approving the form. When the employee applies for ¡§continuing-overtime work form¡¨, the supervisor can also continue to approve the overtime work form.
Numbers of minutes a day to allow late: When the card-brushed on-duty time of an employee exceeds ¡§Latest flexible Sign-in¡¨ in his shift schedule + ¡§Numbers of minutes a day to allow late¡¨ set up in this field, it would then be counted as late at that day.
Maximum to allow late each period: Here ¡§each period¡¨ means the time within the start date and end date of the period as set up in the settlement. If the late arrival count within the period exceeds the setup value of this field, then the late arrival will started to be counted as truancy. (For example: Three late arrival counts are set up to be allowed for each period. When certain person has fourth late arrival at this period, it will start to be seen as truancy.)
Labor hour: This setup value is the basis for the calculation in the leave application form in the duty system. Since the minimum unit of certain leave is half day and all day, hence, for the unification of work hour definition, this setup value is going to be used as calculation basis for work hour.
