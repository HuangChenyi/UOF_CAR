How to approve a View Request

Members who are authorized to apply View Request can only apply while the directory is set as [Allow View Request].

View Request waiting for approval
  Simply Approval
Click [View Request Approval], it will list all requests, you can click the Document Name to view the document.
Check the request and click [Approved] if the request is approved.
Check the request, click [Reject] and input the comments if the request is rejected. The requesters can view the Approver, reason of view request and comments.
  BPM Approval
Click the View Request at [Form Approval] in Homepage to approve the View Request. Click the Document Name to view the document.
Select [Approved] or [Reject] to the View Request.
Click ¡§Confirm¡¨ to finish.

Note: DMS simplified approval mode: If there are several administrators in the directory authorization level, as long as anyone of them has approved, the document can be released.
