Approval form checked out

For the applicant and approver, before the approval is made in the next stop, the approver can take back the approval form because there might be the case that the form needs to be taken back to change the form content or to change the review status.

Use the application date to search form to be checked out, then click ¡§Checked out¡¨ on the form to be Checked out, then click ¡§OK¡¨, the form will then get back to within form to be approved.
Form application void

For the applicant, as long as the application form is not closed, the form can always be cancelled. This is because that the form might be sent out in mistake; or during the form approval process, some problems arise and the form needs to cancelled and a new form needs to be initiated.

Use application date to search form to be cancelled, then click ¡§Void¡¨ on form to be cancelled. You can select ¡§Save to draft area¡¨ or ¡§Void this applied form¡¨.
If you select ¡§Save to draft area¡¨, then you should enter first the name of draft folder, then click ¡§Submit¡¨ to save the form into the draft folder. Before a new version is added for the form, the form can be sent out through this draft folder.
If you select ¡§Void this applied form¡¨, input the reason of void and press "Confirm", the status of this form will then be changed to ¡§Void¡¨. You can see the reason in the flow record.
