Analysis

It lets manager be able to check the analysis of the system so that the manager can control the system better.

System
The last month login statistic: It is the visitor number and flow rate for the login users within one month.
Logout user search: It will list people who did not log in the system before certain date.
Announcement Publish Statistic: The notification category statistics within certain period.
Briefcase capacity statistic: The statistics of all briefcases.
Message statistics within a month: The daily message sending statistics in the recent one month.
