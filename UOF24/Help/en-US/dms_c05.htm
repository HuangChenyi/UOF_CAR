PDF setup

System administrator can set up aiming at PDF watermark part

PDF add in document summary
When the file is converted into PDF format, the system will add automatically abstract of this document into PDF file, therefore, when the user is downloading, the user will see the abstract of this document.

Note:

Document summary means: Author, file, version, confidential grade, valid date, expiry date, announcement date, downloader, downloading time and document downloading frequency, etc.
