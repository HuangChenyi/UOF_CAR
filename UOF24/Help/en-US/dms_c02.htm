Event Logs

Record adding, deleting and revising of each document, hence, in the future, this can be used as basis if the document history is to be tracked.

Checking can be made based on three types, namely, document, directory or safety level ; checking can also be based on certain document type, for example, doc and ppt, etc.
Select the action to be checked, and it can be divided into four types: New, delete, modify and download, for example, add new to the directory and the change of the document.
Select the starting and ending date to be checked and the person to maintain the document, then press ¡§Search¡¨, event record will then be listed.
