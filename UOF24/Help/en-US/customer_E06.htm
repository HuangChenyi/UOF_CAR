How to import and export customer data

When system is to set up or acquire multiple customer data, it can be achieved through importing or exporting customer data. However, only the user with the authority of data manager can use the function of importing and exporting customer data.

Import customer

When system is to set up massive customer data, it can be achieved through importing customer data, which is pretty fast and convenient.

EXCEL contains two SHEETs, one is customer data SHEET, another one is contact person data SHEET.

When it is imported, one customer data can be set up with multiple categories, and categories are separated by¡§;¡¨.

Note: Imported customer data must be set up with data belonging before it can be released.

Export customer

When the system is to acquire massive customer data, it can use exporting data way to acquire customer data, which is fast and convenient.

Press ¡§Export¡¨ at the functional menu on the upper side, a dialogue window for exporting customer will then appear.

Select the data range to be exported. The export information range can be divided into: All customer information and by customer category.

After selecting data range to be exported, press ¡§Submit¡¨, a dialogue window for downloading the file will appear, please click ¡§save¡¨ to get the exported customer data (Including contact person data).

Note: The exported data are all released customer data.
