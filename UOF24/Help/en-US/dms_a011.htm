Directory authorization level setup

Different directory can be set up with different reader, author and administrator, and for different authorization level, different utilizations on the document can be applied.

Click on the ¡§Directory¡¨ tag at the left side the directory to be set up.

Go to the tool bar at the upper side to click ¡§Dir. Control¡¨.

There are 11 directory access controls, namely: Document Setting, Authorization level setup, Confidentiality setup, Approval flow, View request flow, Watermark setup, Document attribute limit, Forced subscription directories, Category setup, Default Text and Document No. setup .
 

Document Setting : Setup File Mode or Text Mode or both, you can also check the option to allow Readers to Query the draft, Print the Text and using the same Document Name in a Directory.
 

Authorization Level Setup : Setup up Administrator, Author & Reader for each Directory.
 

Document Attribute Limit : Setup the required Attribute fields while adding a new document by Authors.
 

Category Setup : Setup usable Categories for Documents, prevent Authors from selecting the wrong Category.
 

Default Text : If the document is Text Mode, you can setup Remarks or default Sample text to remind Author.
 

Setup Document No. : Setup System or Manual Serial No., you can still setup Prefix and format of Serial No. if it is System Serial No.
 

Other Setup : Please refer to Confidentiality setup, approval flow, view request flow, watermark setup and forced subscription directories.
