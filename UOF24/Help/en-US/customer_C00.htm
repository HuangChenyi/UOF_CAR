It records the adding, deletion, modification and release action of customer data, hence, in the future, if customer data needs to be tracked, this can facilitate the query. This function is only available for data manager.

Select the date of action to be queried, and the action by person for maintaining customer data.

Select the action to be queried, which can be divided into four types: New, modify, delete and release, for example, adding to the customer data and modification to the contact person data.

Select the customer data status to be queried, which can be divided into three types: All, released and not released.
