Maintain Flow

Administrator can maintain and manage the electronic approval form of document control system, which includes the announcement of review process and the checking of the review process.

Release approval process¡BView request flow
At the left side of the maintenance of document approval flow, click ¡§Release approval process¡¨.
In the screen, all the approval process will be listed, and you can then see if the flow is in use or not in use, and you can also see which directory uses which flow.
If the flow is not further to be used, you can select to delete the flow, but only the unused flow can be deleted.
The name of each process corresponds to a directory. The flow maintenance person is the setup person of the flow, that is, when a flow is set up at certain directory, it then should be maintained by the original directory administrator.
Each directory can co-use the same flow, hence, if it is to be changed, you can select ¡§Change current directory ¡¨.
