How to search questionnaire

You can search personally prepared questionnaire, and browse the result on return of questionnaire.

Questionnaire search

Enter ¡§Maintain¡¨ screen, then click tree node of questionnaire category on the left side of the screen, then select ¡§Status¡¨, and the default questionnaire status is ¡§Issuing¡¨.

 

¡§Issuing¡¨ means the questionnaire is under preparation.

¡§Questionnaire in progress¡¨ means questionnaire preparation is finished and is activated, and questionnaire survey in undergoing.

¡§Stop¡¨ means the questionnaire is stopped.

¡§End¡¨ means that questionnaire survey has been finished.

Enter ¡§Questionnaire name¡¨ as key word, then click ¡§Query¡¨, then right side of the screen will display questionnaire data list meeting the query conditions.

Browse result on returns

Select the questionnaire to be viewed with result on returns, then click ¡§Results on returns¡¨ at the rear side of questionnaire data, a dialogue box for the Results on Return of the questionnaire will then appear.

Results on Returns can be viewed only when the status of the questionnaire is in progress, stop or end; issuing questionnaire cannot be viewed with the Results on Returns.

Display all selectable answers: When this is checked, then when you browse the data of answered questionnaire, it will then display all the items of the title; if it is not checked, it will only display the items selected by the answerer.

Resend notice to members who have not yet filled in the questionnaire: It will send one more time the questionnaire answering notification for member not answering the questionnaire.

If it is registered questionnaire, ¡§Member name¡¨ of the Results on Returns will display the name of the questionnaire answerer.

If you are to check the content of registered questionnaire, click ¡§Examine content¡¨, a dialogue window for browsing the answered questionnaire content will then appear to display the content answered by the answerer.

If it is anonymous questionnaire, it will only display the words returned questionnaire, and it will not display the name of the questionnaire answerer.

If you want to check anonymous questionnaire content, please click questionnaire name under ¡§Questionnaire name¡¨ field, dialogue window for browsing answered questionnaire content will then appear to display the content of the answered questionnaire.
