Category setup
Click “Maintain Album” in “Video”, then click “Maintain Category” to start setting up category.

Click “New category” to add a new album category. Category is multi-layer tree structure, and category can be arranged according to personal thinking. Click “Move up” and “Move down” to arrange the category order.

Maintain album
New album: In album category, click first category to be saved. Then click “New album” , then enter album topic, album description and to whom the album is to be shared, next, set up photo in album.
If you want to delete album, please check album first, then select “delete album”, it will then be deleted together with the photo.
Move album: Check first the album, then click “Album move” to select which album category it is going to be moved to, the album can then be moved.
Maintain photo
After the setup of album, it’s time to start uploading the photo, please select album to start adding new photo.
Click “New photo” to enter photo maintenance screen. Next, upload the photo, then click “browse” to select photo, then enter the photo subject and brief description of the photo, and if you want to set up the photo as cover, please click “□”.
If you want to delete the photo, please check the photo, then select “Delete photo”.
Move photo: Check the photo first, then select “Photo move”, next select which album it is going to be moved to, and the album can then be moved.

Note: The photo size setup is in “album setup” under “General setup”.
