Form manager management

The manager is authorized to manage each form so as to share the job of system administrator, and BPM can be well divided.

Form manager Mgt.

Select first the form name, then select the form status, and the status can be divided into five types: All, in process, closed, abnormal and void. You should follow the query content to select the status.

Applicant: You can enter name, account and ID number to query the form applicant;
Application time: The application time of the form.

Condition of Form No. : You can use form number as keyword for query, and you can also use the start and end of the form number, finally, click ¡§query¡¨, the system will then list the queried forms, you can also export the list to Excel by clicking "Export".

For the queried forms, different actions will be performed for different situations, and the operations can be divided into 11 types: View content, view flow, print form, void, forced closure, closure restore, unlock, send to next stop, specify approval, follow-up and modify approval result. Each operation will be different depending on the right authorized.

View content, view flow and print form: In the form list, you can select to view and print the form to know the related information of the form.
Void: Since the form has trouble or is filled wrongly, and the form needs to be voided. The voided form can be deleted.
Forced closure: It is needed to check¡§Forced closure allowed¡¨in the advanced setup of ¡§maintain form¡¨ before the operation is activated. Then enter the closure cause to end this form in forced way, and it will be ended for subsequent approval.
Closure restore: You need to chec¡§If closure allowed to return to unclosed¡¨in the advanced setup of ¡§maintain form¡¨ before the operation is activated. When you restore the closed form into unclosed form, it will be restored to the last stop.
Unlock: If it is ¡§multi-or approval¡¨or¡§multi-and approval¡¨form, then when certain approver has read the form but does not approve it, it will lead to the incapability of approval from other approvers. Hence, the administrator can unlock the form so that other approvers can do the approval on it.
Send to next stop: When the approver clicks ¡§abnormal sent¡¨, administrator can click¡§send to next stop¡¨to set up the approval person so that the form can go to the next stop or be closed, and the subsequent stop will become free flow.
Specify approval: When the approver does not approve for a long time, or when the approver forgets to set up the substitute, the administrator can set up for the approver so as to speed up the processing speed.
Follow-up: Enter the follow-up screen, then check the target to be followed up. Next, enter¡§send message¡¨, and the default message is "please approve ASAP", then click¡§Submit¡¨ , the system will then send out private message to the person followed up.
Modify approval result: It is need to check¡§approval result modification allowed¡¨in the advanced setup of ¡§maintain form¡¨ before the operation is activated. Change the form result status from pass to reject, or from reject to pass.
